How to Compile:
->open cmake
->give src where .cxx and CMakeList is present. In my case vtk-proj/ques2/src.
->give build where .cxx and CMakeList is present. In my case vtk-proj/ques2/build.
->press configure
->press generate. 

*** make sure ironProt.vtk is in debug folder after built.
